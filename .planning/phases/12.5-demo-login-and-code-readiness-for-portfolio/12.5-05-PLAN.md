## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-12.5-01 | Elevation of Privilege | demo button reachable in prod | mitigate | Button rendered only when `NEXT_PUBLIC_DEMO_LOGIN_ENABLED === 'true'`; backend (Plan 01) rejects regardless — UI gating is convenience, backend is authoritative |
| T-12.5-03 | Spoofing | JWT session confusion across providers | mitigate | Credentials branch sets the same token fields as Google; no provider field stored in the JWT; session callback unchanged; guards untouched |
| T-12.5-AUTH | Information Disclosure | authorize() throwing leaks 500 / error detail | mitigate | authorize() returns null on every failure path (Pitfall 1) — maps to standard sign-in error, no stack leak |
