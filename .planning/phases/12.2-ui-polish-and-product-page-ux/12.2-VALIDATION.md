## Automated Verification Commands (Copy-Paste Ready)

```bash
# D-01: Collapse defaults
grep -n "useState(false)" \
  nemea-front/src/components/products/ProductNameGroup.tsx \
  nemea-front/src/components/products/ProductFinishGroup.tsx
# Expected: 2 matches

# D-02: TypeGroup unchanged
grep -n "useState(true)" nemea-front/src/components/products/ProductTypeGroup.tsx
# Expected: 1 match

# D-04: Home page is RSC
grep "use client" "nemea-front/src/app/(app)/page.tsx"
# Expected: no output (0 matches)

# D-08: Typography audit clean
grep -rn "font-medium\|font-bold" \
  nemea-front/src/components/ nemea-front/src/app/ \
  --include="*.tsx" --include="*.ts" \
  | grep -v "components/ui/"
# Expected: 0 lines

# D-11: text-right count in ProductFinishGroup
grep -c "text-right" nemea-front/src/components/products/ProductFinishGroup.tsx
# Expected: 6+ (3 headers + 3 cells)

# D-08: font-semibold density (sanity check — should be high)
grep -c "font-semibold" nemea-front/src/components/products/ProductExpandedRow.tsx
# Expected: 11+
```
