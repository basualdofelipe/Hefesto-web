**Part A — Home page rewrite (D-03, D-04):**

Completely rewrite `nemea-front/src/app/(app)/page.tsx`. The new version must:

1. Remove `'use client'` directive — this becomes an RSC (per D-04)
2. Remove ALL current imports (useTheme, useSyncExternalStore, Button)
3. Import: `auth` from `@/auth`, `NO_PERMISSIONS` and `Permissions` type from `@/types/permissions`, `Link` from `next/link`, `Card`/`CardHeader`/`CardTitle` from `@/components/ui/card`, icons from `lucide-react`
4. Make the default export an `async function Home()` (RSC pattern)
5. Read session server-side: `const session = await auth();`
6. Extract permissions: `const permissions: Permissions = session?.user?.permissions ?? NO_PERMISSIONS;`
7. Define 4 card items, each with: label, href, icon, show (boolean from permissions):
   - Productos → `/productos` → ShoppingBag → `permissions.canViewProducts`
   - Calculadora → `/calculadora` → Calculator → `permissions.canUseCalculator`
   - Insumos → `/insumos` → Package → `permissions.canViewSupplies`
   - Gastos → `/finanzas/gastos` → Receipt → `permissions.canViewExpenses`
8. Filter cards to only those where `show` is true
9. Render a responsive grid: `grid grid-cols-2 gap-4 p-6 sm:grid-cols-4`
10. Each card is a `Link` wrapping a `Card` with hover state: `hover:bg-muted/50 cursor-pointer transition-colors`
11. Inside each Card: icon at `size-8 text-primary` + CardTitle at `text-base font-semibold`
12. The function return type must be `Promise<ReactElement>`

Do NOT include a theme toggle. Do NOT include the "NEMEA" heading or subtitle. This is purely quick-link navigation per D-03.
