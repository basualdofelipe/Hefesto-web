## Decisions Made

- Used `await auth()` (server-side) instead of `usePermissions()` hook for home page: D-04 mandates RSC (no `'use client'`), and React hooks cannot run in RSC. The `Permissions` shape from `auth()` is identical to what `usePermissions()` returns — no behavioral difference.
- Only Name and Finish groups default to collapsed — TypeGroup intentionally stays `useState(true)` per D-01 so users see their product types immediately on page load.
