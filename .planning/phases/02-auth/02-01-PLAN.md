    **5. Create JwtAuthGuard** at `nemea-back/src/auth/guards/jwt-auth.guard.ts`:
    - Extend `AuthGuard('jwt')` from `@nestjs/passport`
    - Inject `Reflector` in constructor
    - Override `canActivate()`:
      - Check `IS_PUBLIC_KEY` metadata via reflector (getAllAndOverride on handler + class)
      - If public, return `true` (skip auth)
      - Otherwise, call `super.canActivate(context)`
    - Override `handleRequest<TUser>()`:
      - If error or no user, throw `UnauthorizedException('Token invalido o expirado')`
      - Otherwise return user

    **6. Create RolesGuard** at `nemea-back/src/auth/guards/roles.guard.ts`:
    - Implement `CanActivate` interface
    - Inject `Reflector` in constructor
    - In `canActivate()`:
      - Read `ROLES_KEY` metadata via reflector (getAllAndOverride on handler + class)
      - If no roles required, return true (any authenticated user)
      - Get `user` from `request` via `context.switchToHttp().getRequest()`
      - Check if `user.role` is in required roles
      - If not, throw `ForbiddenException('Permisos insuficientes')`
