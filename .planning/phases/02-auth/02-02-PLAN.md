  <action>
    **1. Create AuthService** at `nemea-back/src/auth/auth.service.ts`:
    - Inject `JwtService` (from `@nestjs/jwt`), `UsersService`, and `ConfigService`
    - In constructor, create `OAuth2Client` from `google-auth-library` with `GOOGLE_CLIENT_ID`
    - Method `validateGoogleToken(idToken: string): Promise<AuthResponseDto>`:
      1. Verify Google id_token: `this.googleClient.verifyIdToken({ idToken, audience: GOOGLE_CLIENT_ID })`
      2. Extract payload: `ticket.getPayload()` — if no payload or no email, throw `UnauthorizedException('Token de Google invalido')`
      3. Check whitelist: `this.usersService.findActiveByEmail(payload.email)` — if null, throw `UnauthorizedException('Usuario no autorizado')`
      4. Update Google profile data: `this.usersService.updateGoogleProfile(user.id, { name: payload.name ?? null, pictureUrl: payload.picture ?? null, googleId: payload.sub })`
      5. Mint JWT: `this.jwtService.sign({ sub: user.id, email: user.email, role: user.role })`
      6. Return `{ accessToken, user: { id, email, role, name, pictureUrl } }`
    - Method `getProfile(userId: number): Promise<User | null>`:
      - Call `this.usersService.findById(userId)`

    **2. Create AuthController** at `nemea-back/src/auth/auth.controller.ts`:
    - `@Controller('auth')` with `@ApiTags('auth')`
    - Add `@ApiBearerAuth()` at class level for Swagger
    - Endpoint `POST /auth/google`:
      - `@Public()` — this endpoint is called BEFORE the user has a backend JWT
      - `@HttpCode(HttpStatus.OK)` — return 200, not 201
      - Accept `@Body() dto: GoogleAuthDto`
      - Call `authService.validateGoogleToken(dto.idToken)`
      - Return the result (ResponseInterceptor wraps in `{ data: ... }`)
      - Swagger: `@ApiOperation({ summary: 'Exchange Google id_token for backend JWT' })`, `@ApiResponse({ status: 200, type: AuthResponseDto })`
    - Endpoint `GET /auth/me`:
      - Uses `@CurrentUser()` to get the authenticated user from the JWT
      - Calls `authService.getProfile(user.id)` to get full user data from DB
      - If not found, throw `UnauthorizedException()`
      - Return user profile
      - Swagger: `@ApiOperation({ summary: 'Get authenticated user profile' })`, `@ApiResponse({ status: 200 })`

    **3. Update AuthModule** at `nemea-back/src/auth/auth.module.ts`:
    - Add `AuthService` to providers
    - Add `AuthController` to controllers
    - Add `AuthService` to exports (optional, but useful for future modules)
