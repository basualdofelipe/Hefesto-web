**2. Create usePermissions() hook** (`nemea-front/src/hooks/usePermissions.ts`):
Per D-12 and D-13.
```typescript
'use client';

import { useSession } from 'next-auth/react';
import type { Permissions } from '@/types/permissions';
import { NO_PERMISSIONS } from '@/types/permissions';

export function usePermissions(): Permissions {
  const { data: session } = useSession();
  return session?.user?.permissions ?? NO_PERMISSIONS;
}
```

**3. Update NextAuth type declarations** (`nemea-front/src/types/next-auth.d.ts`):
Replace `role: string` with `permissions: Permissions` in the Session.user interface. Add `permissions` to the JWT interface. Keep `accessToken`, `id`, `email`, `name`, `image`:
```typescript
import 'next-auth';

    permissions?: Permissions;
    userId?: string;
  }
}
```
Remove `role?: string` from JWT. Remove `role: string` from Session.user.

**4. Update auth.ts callbacks**:

**5. Update middleware.ts** (per D-14):
Replace `ADMIN_ONLY_ROUTES` array with a `ROUTE_PERMISSIONS` map. Each route prefix maps to the permission required to access it:
```typescript
import type { Permissions } from '@/types/permissions';

const ROUTE_PERMISSIONS: Record<string, keyof Permissions> = {
  '/catalogos': 'canViewProducts',
  '/proveedores': 'canViewSupplies',
  '/insumos': 'canViewSupplies',
  '/productos': 'canViewProducts',
  '/finanzas': 'canViewExpenses',
  '/usuarios': 'canManageUsers',
  '/configuracion': 'canManageConfig',
  '/calculadora': 'canUseCalculator',
  '/escenarios': 'canManageScenarios',
  '/roles': 'canManageUsers',
};
```

Replace the `isAdminOnlyRoute` function with a `getRequiredPermission` function:
```typescript
function getRequiredPermission(pathname: string): keyof Permissions | null {
  for (const [route, permission] of Object.entries(ROUTE_PERMISSIONS)) {
    if (pathname === route || pathname.startsWith(`${route}/`)) {
      return permission;
    }
  }
  return null;
}
```

Update the middleware auth callback:
- Instead of checking `req.auth.user?.role !== 'admin'`, check:
```typescript
const requiredPermission = getRequiredPermission(pathname);
if (requiredPermission && req.auth) {
  const permissions = req.auth.user?.permissions;
  if (!permissions || !permissions[requiredPermission]) {
    return Response.redirect(new URL('/', req.nextUrl.origin));
  }
}
```
- Remove the `ADMIN_ONLY_ROUTES` array entirely.
- Keep the unauthenticated redirect and the backendToken check unchanged.
