## What Was Built

Created `nemea-front/src/constants/branding.ts` — modulo flat de named exports (mismo shape que `tiendanube.ts` sibling):

```typescript
// NEXT_PUBLIC_* se inlinean en build time — rebuild requerido para cambios.
export const APP_NAME = process.env.NEXT_PUBLIC_APP_NAME ?? 'Hefesto';
export const DEMO_EMAIL =
  process.env.NEXT_PUBLIC_DEMO_EMAIL ?? 'demo@hefesto.com';
export const CONTACT_EMAIL =
  process.env.NEXT_PUBLIC_CONTACT_EMAIL ?? 'admin@hefesto.com';
```

Migraciones:
- `layout.tsx`: `title: 'Nemea'` → `title: APP_NAME`
- `login/page.tsx`: `alt='NEMEA'` → `alt={APP_NAME}`
- `AppSidebar.tsx`: ambos `alt='NEMEA'` (lineas 83, 90) → `alt={APP_NAME}`
- `DemoLoginButton.tsx`: const local `'demo@nemea.app'` eliminada → importa `DEMO_EMAIL` de branding
- `acceso-denegado/page.tsx`: heading `NEMEA` → `{APP_NAME}`, texto `Nemea` → `{APP_NAME}`, contacto `basualdofelipe@gmail.com` → `{CONTACT_EMAIL}`
