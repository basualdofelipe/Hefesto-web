## Sampling Rate

- **After every task commit:** Run the corresponding quick-run command (backend or frontend, scoped to the touched module).
- **After every plan wave:** Run the full suite for the affected app + type check.
- **Before `/gsd:verify-work`:** Full suite (both apps) must be green, plus both lints, plus both `tsc --noEmit`.
- **Max feedback latency:** 30 seconds for quick runs; ~90 seconds for full pre-verify.
