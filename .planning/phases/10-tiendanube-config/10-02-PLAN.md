<objective>
Build the frontend config page at /configuracion/tiendanube where admin can view and edit all Tiendanube rate tables including plan CPT rates. Includes plan selector dropdown at the top per CONTEXT.md, a new PlansSection for CPT editing, and sidebar navigation link. This fulfills TNCF-01 through TNCF-04.

Purpose: Admin needs a UI to manage Tiendanube rates that feed the calculadora. Without this page, rates can only be changed via direct DB access. The plan CPT editing (TNCF-03) is critical because the transaction fee per plan affects all pricing calculations.
Output: Working config page with plan selector, collapsible sections for gateways/plans/installments/taxes, editable rates, and "Verificar tasas" external link.
</objective>

GET /api/tiendanube-config/all returns:
```typescript
interface TiendanubeConfigAll {
  gateways: TnPaymentGateway[];
  rates: TnGatewayRate[];       // latest per (gateway, method, days) combo
  installments: TnInstallmentRate[];  // latest per installment count
  taxConfig: TnTaxConfig;        // latest single row
  plans: TnPlan[];
}

interface TnPaymentGateway {
  id: string;
  slug: string;       // 'pago_nube' | 'mercado_pago' | 'modo'
  label: string;      // 'Pago Nube' | 'Mercado Pago' | 'MODO'
  isActive: boolean;
  createdAt: string;
}

interface TnGatewayRate {
  id: string;
  gateway: TnPaymentGateway;  // nested relation
  paymentMethod: string;       // 'tarjeta_debito_credito', 'billetera_virtual', etc.
  withdrawalDays: number;
  ratePercent: number;         // PARSED to number by backend service (not string)
  isActive: boolean;
  createdAt: string;
}

interface TnInstallmentRate {
  id: string;
  installments: number;  // 1, 3, 6, 9, 12
  ratePercent: number;   // PARSED to number by backend service
  isActive: boolean;
  createdAt: string;
}

interface TnTaxConfig {
  id: string;
  ivaRate: number;     // PARSED to number by backend service
  iibbRate: number;    // PARSED to number by backend service
  isActive: boolean;
  createdAt: string;
}

interface TnPlan {
  id: string;
  slug: string;             // 'inicial', 'esencial', 'impulso', 'escala'
  label: string;            // 'Inicial', 'Esencial', 'Impulso', 'Escala'
  cptPagoNube: number;      // PARSED to number by backend service
  cptOtherGateways: number; // PARSED to number by backend service
  onlyPagoNube: boolean;
  isActive: boolean;
}
```
